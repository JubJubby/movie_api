URL endpoints:
/movies returns a list of all movies using a GET request.
/movies/movieName returns a movie's data by the movie's name using a GET request (use "%20" in place of a space).
/movies/genre/genreName returns movies by the movies' genre using a GET request.
/movies/director/directorName returns director's data by name using a GET request.
/users creates a new user using a POST request.
/users/ID updates a user's name using a PUT request.
/users/ID/movieName adds a movie to a user's favorites list using a POST request.
/users/ID/movieName removes a movie from a user's favorites list using a DELETE request.
/users/ID removes a user using a DELETE request.
